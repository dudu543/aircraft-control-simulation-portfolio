# Aircraft 6-DOF Simulation

## 1. Background

本项目用于实现飞行器刚体六自由度动力学仿真，展示 NED 坐标系、机体系转换、姿态角响应、速度响应和位置响应的基本建模能力。

## 2. Model

计划建立包含位置、速度、欧拉角、角速度的刚体六自由度模型。模型采用 NED 地面坐标系和机体坐标系，通过欧拉角转换矩阵实现坐标变换。

## 3. Method

使用 Python 或 MATLAB 进行数值积分，输入简化外力和外力矩，输出飞行器状态响应。

## 4. Simulation Cases

- 姿态扰动响应
- 恒定力输入下的位置和速度响应
- 简化力矩输入下的滚转/俯仰/偏航响应
