# eVTOL Vertical Takeoff under Wind Disturbance

## 1. Background

本项目基于本科毕业设计内容，整理 eVTOL 在城市风场干扰下的垂直起降控制仿真，展示飞行动力学建模、风场建模、双环控制律设计和仿真验证能力。

## 2. Model

模型包括 eVTOL 垂直起降阶段动力学、螺旋桨拉力模型、机体气动力模型和城市风场模型。风场由平均风、阵风和随机风叠加构成。

## 3. Method

采用外环位置控制、内环姿态控制的双环 PID/PD 控制结构，对比无风、有风和控制器参数优化后的飞行响应。

## 4. Simulation Cases

- 无风垂直起降仿真
- 有风干扰垂直起降仿真
- 控制器参数优化后仿真
- 高度响应、位置误差、姿态角和控制量对比
