# Proportional Navigation Guidance Simulation

## 1. Background

本项目用于实现二维导弹-目标拦截场景下的比例导引律仿真，展示导航制导算法建模、仿真和结果分析能力。

## 2. Model

建立二维平面内导弹与目标的相对运动模型，目标可设置为匀速直线运动或简单机动运动。

## 3. Method

实现 PN 比例导引律，根据视线角速度生成法向加速度指令，分析不同导航比、初始交会角和目标机动条件下的拦截效果。

## 4. Simulation Cases

- 目标匀速直线运动
- 目标机动运动
- 不同初始交会角
- 不同导航比 N
- 脱靶量对比
