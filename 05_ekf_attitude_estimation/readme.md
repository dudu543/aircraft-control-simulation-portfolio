# EKF Attitude Estimation

## 1. Background

本项目用于实现基于 IMU 仿真数据的扩展卡尔曼滤波姿态估计，展示飞行器状态估计和传感器数据处理能力。

## 2. Model

生成包含陀螺仪角速度和加速度计观测的仿真 IMU 数据，加入噪声和偏置，用于姿态估计算法验证。

## 3. Method

实现 EKF 姿态估计流程，包括状态预测、观测更新、误差协方差传播和滤波结果分析。

## 4. Simulation Cases

- 无噪声姿态估计
- 带噪声 IMU 数据姿态估计
- 加速度计/陀螺仪噪声对比
- 滤波前后姿态误差对比
